# Общество — TV App

## ⚠️ ВАЖНО: localStorage НЕ подходит для реального зала

`cloudEnabled: false` работает **только на одном устройстве** (разные вкладки одного браузера).

**Для реального зала (телефон администратора + отдельный телевизор) обязательно нужен Google Script.**

Без Google Script телефон и телевизор не видят друг друга.

---

## Быстрый старт — тест на одном компьютере

1. Распаковать zip
2. В `src/core/config.js` оставить `cloudEnabled: false`
3. Открыть в **разных вкладках одного браузера**:
   - `app/admin-timer.html`
   - `app/admin-seating.html`
   - `app/tv.html`
4. Нажать СТАРТ в admin-timer → переключить TV на таймер → тикает ✓

---

## Реальный зал — подключение Google Script

### Шаг 1 — Создать скрипт

1. Открыть [script.google.com](https://script.google.com)
2. Создать новый проект
3. Вставить содержимое `google-script/Code.gs`
4. В строке `const TOKEN = '...'` придумать секретный токен

### Шаг 2 — Задеплоить

1. **Развернуть** → **Новое развёртывание**
2. Тип: **Веб-приложение**
3. Выполнять от имени: **Я**
4. Доступ: **Все**
5. Нажать **Развернуть**, скопировать URL

### Шаг 3 — Прописать в config.js

```js
const APP_CONFIG = {
  cloudEnabled: true,   // ← ОБЯЗАТЕЛЬНО true для реального зала
  cloudUrl: 'https://script.google.com/macros/s/ВАШ_ID/exec',
  cloudToken: 'ВАШ_ТОКЕН',  // ← тот же что в Code.gs
  cloudTimeout: 6000,
  env: 'prod'
};
```

### Шаг 4 — Проверить

Открыть в браузере:
```
https://script.google.com/macros/s/ВАШ_ID/exec?path=timer-state&token=ВАШ_ТОКЕН
```
Должен вернуть `null` (нет данных) — значит работает.

## Архитектура

| Файл | Роль |
|------|------|
| `admin-timer.html` | Единственный **writer** таймера |
| `admin-seating.html` | Единственный **writer** рассадки |
| `tv.html` | **Read-only** — только читает |

TV никогда не пишет в cloud.

---

## Аварийный режим

Если Google Script упал или нет интернета — в `admin-timer.html` нажать **🆘 ЛОКАЛЬНЫЙ РЕЖИМ**.
Таймер продолжит работать только на этом устройстве.
Нажать ещё раз — вернётся cloud-режим.

## Структура блайндов (утвердить перед игрой)

`src/timer/structures.js` — два варианта:
- **default** (Обычная / Суббота): 25 уровней, анте с уровня 7 = SB
- **action30k** (Экшен 30К): 23 уровня, анте с уровня 3 = SB

**Перед реальным запуском сверить:**
- [ ] с какого уровня анте
- [ ] анте = SB или другое значение
- [ ] уровни 11–15 (после адд-она)
- [ ] финальные уровни

---

## Чек-лист перед реальным залом

- [ ] `cloudEnabled: true`
- [ ] URL и TOKEN вставлены в `config.js`
- [ ] Тот же TOKEN в `google-script/Code.gs`
- [ ] Google Script задеплоен как Web App (доступ: Все)
- [ ] GET-запрос к Script вернул корректный JSON
- [ ] Тест: телефон (admin-timer) + другое устройство (tv.html) — таймер синхронизируется
- [ ] Структура блайндов сверена вручную
